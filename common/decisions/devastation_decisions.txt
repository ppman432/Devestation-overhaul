# =====================================================================
# BETTER MECHANICS DEVASTATION — Decisions
# =====================================================================
# Warfare Conduct: 7 stances controlling military behavior toward civilians
# Refugee Policy: 5 stances controlling how open the country is to refugees
#
# Mechanics:
# - Each decision uses days_remove = -1 so it stays active permanently
# - visible = always so active stance stays visible (greyed out = currently active)
# - available = NOT { has_country_flag = bm_X_active } gates re-selection
# - Dynamic modifiers applied/removed on complete_effect
# - warfare_stance_civ_mult: multiplies occupation casualty formula
# - warfare_stance_dev_mult: multiplies devastation gain in devastation_state_calc_value
# - Opinion modifiers applied globally for aggressive stances
# - One-time stability/war_support effects on complete_effect
# =====================================================================

			set_variable = { warfare_stance_dev_mult = 0.70 }
